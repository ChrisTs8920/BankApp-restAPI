Through browser:

  • Create new account.
  • View all accounts (xml format)
  • View all accounts (json format)

  • Using this link you can view an account based on id (xml format): http://localhost:8080/war_exploded/api/bankService/allAccounts_xml/{aid}

  • Using this link you can view an account based on id (json format): http://localhost:8080/war_exploded/api/bankService/allAccounts_json/{aid}

Through Postman:

  • Deposit in account: PUT http://localhost:8080/war_exploded/api/bankService/deposit/{aid}

  • Withdraw from account: PUT http://localhost:8080/war_exploded/api/bankService/withdraw/{aid}

  • Transfer from account to account: PUT http://localhost:8080/war_exploded/api/bankService/transferBalance/{aid1}?{aid2}=&{amount}=

  • Activate/Deactivate account: PUT http://localhost:8080/war_exploded/api/bankService/updateAccountStatus/{aid}/{status}

  • Delete account: DELETE http://localhost:8080/war_exploded/api/bankService/delete/{aid}

Info: Replace placeholders in {} with the corresponding value.
Example: Replace {aid} with an existing id.
